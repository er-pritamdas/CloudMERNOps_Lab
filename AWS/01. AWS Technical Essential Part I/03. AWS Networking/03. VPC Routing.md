A **route table** is a set of **rules (routes)** that determine **how network traffic is directed** within a VPC.

Each route has:

* A **destination CIDR block** (e.g., `0.0.0.0/0`)
* A **target** (e.g., `igw-xxxx`, `nat-gateway`, `local`, etc.)
